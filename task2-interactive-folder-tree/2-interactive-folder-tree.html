How to use interactive DOM tree

  1. To use expand/collapse feature you should do double click on every folder name, which you would like to expand/collapse. If there are no items to expand/collapse - nothing will happen
  2. To use single selection of folder or subfolder you should press 'Ctrl' key and then press left mouse button. Selected item will be highlighted with lightcoral color
  3. To use multiselection of folders or subfolders you should press 'Shift' key and then press left mouse button. Selected item(s) will be highlighted with lightseagreen color
